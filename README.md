# Lógica de Programação e Algoritmos

Bem-vindo ao meu repositório de estudos! 🚀

Este espaço é dedicado a documentar a minha jornada de reconstrução dos fundamentos da computação. O objetivo aqui não é criar um sistema complexo, mas sim treinar a base lógica pura antes de avançar para a engenharia de software corporativa (com foco em Java e no setor bancário).

# 🎯 Objetivo

Registrar a resolução de problemas lógicos, indo desde estruturas sequenciais básicas até a manipulação de matrizes e vetores, utilizando uma abordagem "sem atalhos" e sem o uso de assistentes de Inteligência Artificial para a escrita de código.

# 📚 Base de Estudos

Os exercícios e conceitos aplicados aqui acompanham o currículo do curso "Algoritmos e Lógica de Programação" do professor Nélio Alves, complementando as disciplinas de "Lógica e Técnicas de Programação" e "Estruturas de Dados" da minha graduação em Análise e Desenvolvimento de Sistemas (Uniasselvi).

# 🛠️ Tecnologias Utilizadas

VisualG / Portugol: Utilizado na fase inicial para focar puramente no raciocínio estruturado, sem a sobrecarga sintática de linguagens complexas.

(Futuro) Java: Evolução natural do repositório para aplicação de Programação Orientada a Objetos (POO).
